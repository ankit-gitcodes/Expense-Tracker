# Expense Tracker

A simple command-line Expense Tracker built with Python.
This application allows users to add expenses, view saved expenses, and calculate the total amount spent.

## 🚀 Features

* Add new expenses
* View all saved expenses
* Calculate total expenses
* Store expense data in a file
* Simple menu-driven interface

## 🛠️ Technologies Used

* Python
* File Handling
* Loops and Conditions
* Exception Handling
* Function

## ▶️ How to Run

1. Make sure Python is installed
2. Open terminal in project folder
3. Run the program:

   ```bash
   python expense_tracker.py
   ```

## 📌 Example Menu

```text
1. Add Expense
2. View Expenses
3. Show Total
4. Exit
```

## 📂 File Storage

Expenses are stored in:

```text
expenses.txt
```

Example format:

```text
Food: 200
Travel: 500
```
